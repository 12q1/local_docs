- **A value used when none has [[b/_been|been]] given; a tentative value or standard that is presumed.**
	- _Example: If you don't specify a number of items, the default is 1._
- **The failure of a [[d/_defendant|defendant]] to [[a/_appear|appear]] and [[a/_answer|answer]] a summons and [[c/_complaint|complaint]].**
