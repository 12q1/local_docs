# Dependency
---
#noun
- **A state of [[d/_dependence|dependence]]; a refusal to exercise initiative.**
- **Something [[d/_dependent|dependent]] on, or subordinate to, something else:**
- **A [[c/_colony|colony]], or a territory subject to rule by an external power.**
- **A [[d/_dependence|dependence]] on a habit-forming substance such as a drug or [[a/_alcohol|alcohol]]; [[a/_addiction|addiction]].**
- **Reliance on the functionality provided by some other, external [[c/_component|component]].**
	- _Example: This library has dependencies on a lot of other libraries. We have to compile all of those other libraries first._
- **An external [[c/_component|component]] whose functionality is relied on.**
	- _Example: One of this library's dependencies is very finicky. It can be hard to get it to compile on some systems._
