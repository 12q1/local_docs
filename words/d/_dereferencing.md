# Dereferencing
---
#verb
- **To [[a/_access|access]] the value or object located in a memory location stored in a pointer or [[a/_another|another]] value interpreted as such; to [[a/_access|access]] a value [[b/_being|being]] referenced by something [[e/_else|else]].**
	- _Example: Attempting to dereference a null pointer often results in a crash._
- **(in operating systems [[d/_derived|derived]] from Unix) To operate on the file or [[d/_directory|directory]] pointed to by a symbolic link, rather than on the link itself.**
	- _Example: With the find command, any symbolic links appearing after -follow will be dereferenced._
---

#noun
- **The [[a/_act|act]] by which something is [[d/_dereferenced|dereferenced]].**
