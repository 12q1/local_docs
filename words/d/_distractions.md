# Distractions
---
#noun
- **Something that [[d/_distracts|distracts]].**
	- _Example: Poking one's eye is a good distraction from a hurting toe._
- **The process of [[b/_being|being]] [[d/_distracted|distracted]].**
	- _Example: We have to reduce distraction in class if we want students to achieve good results._
- **Perturbation; [[d/_disorder|disorder]]; [[d/_disturbance|disturbance]]; [[c/_confusion|confusion]].**
- **Mental [[d/_disorder|disorder]]; a [[d/_deranged|deranged]] state of mind; insanity.**
	- _Example: The incessant nightmares drove him to distraction._
- **Traction so exerted as to separate surfaces normally opposed.**
