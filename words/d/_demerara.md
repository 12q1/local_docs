# Demerara
---
#noun
- **A type of natural, unrefined or partially refined [[c/_cane|cane]] sugar, which is light [[b/_brown|brown]] in [[c/_colour|colour]], particularly used in pastries and [[b/_biscuits|biscuits]] like shortbread.**
- **A [[d/_dark|dark]] rum, made in Guyana using molasses and this sugar, mainly used for [[b/_blending|blending]].**
