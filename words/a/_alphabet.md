# Alphabet
---
#noun
- **The set of letters used when writing in a language.**
	- _Example: In the first year of school, pupils are taught to recite the alphabet._
- **A writing system in which letters represent phonemes. ([[c/_contrast|Contrast]] e.g. logography, a writing system in which each [[c/_character|character]] represents a word, and syllabary, in which each [[c/_character|character]] represents a syllable.)**
- **A typically finite set of distinguishable symbols.**
	- _Example: Let L be a regular language over the alphabet \Sigma._
- **An individual letter of an [[a/_alphabet|alphabet]]; an [[a/_alphabetic|alphabetic]] [[c/_character|character]].**
- **The simplest rudiments; elements.**
