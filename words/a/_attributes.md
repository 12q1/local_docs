# Attributes

#noun
- **A [[c/_characteristic|characteristic]] or quality of a thing.**
	- _Example: His finest attribute is his kindness._
- **([[g/_grammar|grammar]]) A word that qualifies a [[n/_noun|noun]], a qualifier.**
	- _Example: In the clause "My jacket is more expensive than yours", "My" is the attribute of "jacket"._
- **That which is predicated or [[a/_affirmed|affirmed]] of a subject; a predicate; an [[a/_accident|accident]].**
- **An [[o/_option|option]] or setting [[b/_belonging|belonging]] to some [[o/_object|object]].**
	- _Example: A file with the read-only attribute set cannot be overwritten._
- **A semantic [[i/_item|item]] with which a [[m/_method|method]] or [[o/_other|other]] [[c/_code|code]] [[e/_element|element]] [[m/_may|may]] be [[d/_decorated|decorated]].**
	- _Example: Properties can be marked as obsolete with an attribute, which will cause the compiler to generate a warning if they are used._
- **A [[n/_numeric|numeric]] value representing the [[c/_colours|colours]] of part of the screen [[d/_display|display]].**
