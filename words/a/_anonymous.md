# Anonymous
---
#adjective
- **Lacking a name; not named, for example an [[animal]] not [[assigned]] to [[any]] species.**
- **Without [[any]] name [[acknowledged]] of a person responsible**
	- _Example: an anonymous pamphlet_
- **Of unknown name; whose name is withheld**
	- _Example: No customer personal data will be retained unless it is rendered anonymous._
- **Lacking individuality.**
	- _Example: an anonymous office block in a soulless industrial estate_
