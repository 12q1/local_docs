# Arithmetic
---
#noun

#adjective
- **Of, relating to, or using [[a/_arithmetic|arithmetic]]; [[a/_arithmetical|arithmetical]].**
	- _Example: arithmetic geometry_

	- _Example: arithmetic progression_
