# Breakpoints
---
#noun
- **A point in a program at which operation may be interrupted during [[d/_debugging|debugging]] so that the state of the program at that point [[c/_can|can]] be investigated.**
- **Location referred to when issuing watches, warnings, or [[a/_advisories|advisories]] for specific [[a/_areas|areas]].**
- **The point where surface water waves are [[b/_breaking|breaking]] in e.g. oceans or lakes.**
- **A particular size at which a web page is [[d/_designed|designed]] to switch to a different layout.**
