# Environment
---
#noun
- **The surroundings of, and influences on, a particular item of interest.**
- **The natural world or [[e/_ecosystem|ecosystem]].**
- **[[a/_all|All]] the [[e/_elements|elements]] that [[a/_affect|affect]] a system or its inputs and outputs.**
- **A particular political or social setting, [[a/_arena|arena]] or [[c/_condition|condition]].**
- **The software and/or hardware existing on [[a/_any|any]] particular [[c/_computer|computer]] system.**
	- _Example: That program uses the Microsoft Windows environment._
- **The [[e/_environment|environment]] of a function at a point [[d/_during|during]] the execution of a program is the set of identifiers in the function's scope and their [[b/_bindings|bindings]] at that point.**
- **The set of variables and their values in a namespace that an operating system [[a/_associates|associates]] with a process.**
