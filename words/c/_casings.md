# Casings
---
#noun
- **That which encloses or encases.**
	- _Example: Some people like to split the casing of a sausage before cooking so it doesn't burst._
- **The [[decorative]] trim [[around]] a door or window.**
- **A metal pipe used to line the [[borehole]] of a well.**
- **The [[collective]] states of upper and lower [[case]] letters.**
	- _Example: The replacement string should have the same casing as the matched text._
