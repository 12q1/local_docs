# Commute
---
#verb
- **To exchange substantially; to [[a/_abate|abate]] but not [[a/_abolish|abolish]] [[c/_completely|completely]], a penalty, obligation, or payment in return for a great, single thing or an [[a/_aggregate|aggregate]]; to [[c/_cash|cash]] in; to lessen**
	- _Example: to commute tithes into rentcharges for a sum; to commute market rents for a premium, to commute daily fares for a season ticket_
- **Of an operation, to be [[c/_commutative|commutative]], i.e. to have the property that [[c/_changing|changing]] the order of the operands does not [[c/_change|change]] the result.**
	- _Example: A pair of matrices share the same set of eigenvectors if and only if they commute._
