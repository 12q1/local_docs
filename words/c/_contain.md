# Contain
---
#verb
- **To hold inside.**
- **To include as a part.**
- **To put constraint upon; to restrain; to confine; to keep within bounds.**
	- _Example: I'm so excited, I can hardly contain myself!_
- **(of a set etc.) To have as an element or subset.**
	- _Example: A group contains a unique inverse for each of its elements._
- **To restrain desire; to live in continence or chastity.**
